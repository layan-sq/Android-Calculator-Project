# Simple Android Calculator 📱

A functional and user-friendly calculator application developed for Android devices. This project demonstrates my understanding of mobile app logic and UI/UX basics.

##  Key Features
- **Basic Arithmetic**: Supports addition, subtraction, multiplication, and division.
- **Input Validation**: Includes logic to prevent errors, such as dividing by zero.
- **Clean Design**: A simple layout focused on ease of use.

##  Tools & Technologies
- **Language**: Java
- **Interface**: XML (Android Layouts)
- **Environment**: Android Studio

##  Logic Highlight
In this project, I focused on ensuring that the application handles operations correctly and remains stable even with unexpected user inputs.
